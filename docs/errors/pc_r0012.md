# PC_R0012

**Level:** error (thrown, development only) · **Area:** runtime

## What happened

The `PiniaColada` plugin was installed but no Pinia instance was detected. Pinia Colada is built on top of Pinia and needs it to be installed first.

## How to fix it

Install Pinia **before** `PiniaColada`:

```ts
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import { PiniaColada } from '@pinia/colada'

const app = createApp(App)

// ✅ pinia first
app.use(createPinia())
app.use(PiniaColada)
```

If you manage Pinia yourself, pass the instance explicitly:

```ts
app.use(PiniaColada, { pinia })
```

## Common causes

- Registering `PiniaColada` before `createPinia()`.
- Forgetting to call `app.use(createPinia())` entirely.
