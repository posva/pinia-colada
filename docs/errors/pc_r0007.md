# PC_R0007

**Level:** error (development only) · **Area:** runtime

## What happened

The mutation cache `ref` was **reassigned** instead of being mutated in place. The cache must keep the same underlying object so Pinia Colada and its plugins can track entries. Reassigning it would break in production.

## How to fix it

Mutate the existing cache through its actions rather than swapping out the internal map.

```ts
const mutationCache = useMutationCache()

// ❌ replaces the reactive value
// mutationCache.caches.value = new Map()

// ✅ use the public actions instead
```

## Common causes

- Resetting the cache with an assignment instead of the provided actions.
- A devtools or test helper assigning to the internal ref.
